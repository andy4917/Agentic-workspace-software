# Codex Harness Reports

Harness reports and templates for repo verification, context inspection,
retrieval, eval, benchmark, P0 integrity, and audit work live here. `README.md`,
templates, and seed discovery files are active managed source.

`*.latest.json`, `*.latest.md`, and `*results.jsonl` are ignored
runtime outputs. Use them for triage only; rerun the responsible command before
treating a check as current validation. Keep not-run and failed checks explicit.
Do not recreate retained dated evidence archives by default.
