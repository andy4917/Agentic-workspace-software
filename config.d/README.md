# Managed Config Fragments

This directory stores public-safe managed-source fragments for the live Codex
runtime configuration. The active runtime truth remains
`C:\Users\anise\.codex\config.toml`; these fragments are reviewed here, copied
to `C:\Users\anise\.codex\config.d`, then reconciled into `config.toml`.

Do not store secrets here. Use environment-variable references for credentials.
