# Patched Plugin Snapshots

This directory contains historical or staged plugin snapshots only.

Do not use files under `plugins/patched` as live runtime evidence. Current
plugin runtime state must be verified from `%USERPROFILE%\.codex\plugins\cache`,
the active `config.toml`, plugin manifests, health scripts, or fresh tool calls.

Before trusting, syncing, or deleting a snapshot here, classify it as active
managed source, historical evidence, source template, runtime output,
quarantine candidate, or contamination candidate.
