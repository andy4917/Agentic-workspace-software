# Naming Convention

Use stable, descriptive names tied to the managed surface.

- Root name: `Dev_Codex_App_GlobalSSOT`
- Declarative config: `*.agent.config.toml` or `*.agent.config.yaml`
- Hook config: `*.yaml`
- Runtime state: `*.json`
- Shared hook runner: `codex-ssot-hook.ps1`

Windows user-facing reports use `folder` and `PowerShell`.
